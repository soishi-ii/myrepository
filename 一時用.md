

| 単語     | コマンド                             |
| ------ | -------------------------------- |
| dfコマンド | ファイルシステムの空き容量を確認(Mは1,048,576バイト) |
| -h     | 容量を適当な単位で表示する                    |
| -H     | 容量を適当な単位で表示する(Mは1,000,000バイト)    |
| -k     | 容量をKB単位で表示する                     |
|        |                                  |
